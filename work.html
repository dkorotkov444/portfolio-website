REEL Movie App

REEL Movie App is a full‑stack movie discovery application — a React front end consuming an Express + MongoDB REST API — that lets users browse, search, view details and save favorite movies.

Objective: Demonstrate a production‑like full‑stack workflow by building an interactive, responsive UI backed by a secure REST API with JWT user authentication and persistent favorite lists.

Live demo Frontend (GitHub) Backend (GitHub)
Technologies used
  • React
  • Node.js
  • Express
  • MongoDB
  • Mongoose
  • Passport (JWT)
  • bcrypt
  • TMDB / curated dataset
  • express-validator
  • HTML/CSS

Meet Event Finder

Meet is a serverless progressive web app (PWA) built with React and Vite that helps users discover upcoming events across multiple cities, view details, and install the app for offline use.

Objective: Demonstrate PWA features, TDD practices, and secure Google OAuth integration to fetch calendar events.

Live demo Source (GitHub)
Technologies used
  • React
  • Vite
  • JavaScript
  • PWA / Service Worker
  • Google Calendar API (OAuth)
  • Jest
  • React Testing Library
  • HTML/CSS

Pokedex

A small static front-end app that fetches data from the public PokéAPI and displays a paginated list of Pokémon with a details modal and basic navigation.

Objective: Provide a lightweight, dependency-free UI that demonstrates API consumption, modals, and pagination while remaining easy to run without a build system.

Live demo Source (GitHub)
Technologies used
  • Vanilla JavaScript
  • Bootstrap 5
  • PokéAPI
  • Promise & fetch polyfills
  • HTML / CSS
